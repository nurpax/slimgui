# Introduction

The Slimgui package provides modern Python bindings for the following libraries:

- [Dear ImGui](https://github.com/ocornut/imgui)
- [ImPlot](https://github.com/epezent/implot)

Features:

- API design that stays close to Dear ImGui while remaining natural to use from Python
- Python bindings for Dear ImGui and ImPlot with `.pyi` typing files for good IDE support
- Integrations for GLFW + PyOpenGL and Pyglet
